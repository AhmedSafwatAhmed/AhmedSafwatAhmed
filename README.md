### [K_DemoModularBMS](https://github.com/AhmedSafwatAhmed/K_DemoModularBMS) — Modular Battery Management System

**Self-directed portfolio project | FreeRTOS · NXP KL25Z · C**

A hands-on BMS implementation for electric vehicles, built to bridge theoretical knowledge with practical embedded systems skills.

| Module | Technology | Description |
|--------|-----------|-------------|
| **SOC Estimation** | Extended Kalman Filter | State-of-Charge algorithm |
